Red Panda Runner

A high-speed, intuitive compilation runner extension constructed directly for VS Code.

Extension Overview

Red Panda Runner makes local script execution fast and reliable in VS Code. It bridges the gap between writing code and testing it, providing customized runtime hooks and compiler tooling options directly inside your workspace sidebar.

Key Features

✦

Instant Environment Configurations

Provides auto-detection paths for your compilers (such as GCC/G++ or Dart runtimes), helping you skip tedious configurations.

✦

Lightweight Execution Handles

Dedicated commands and hotkey handles to trigger builds or run files natively without cluttering your system command shell.

✦

Tailored Programming UX

Smooth Visual Studio Code interface overlays, built-in debug trackers, and speed analytics designed to improve workflow speed.
